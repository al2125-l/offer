#  Personalized Offer Recommendation with Explainability

##  Project Overview
This project builds a machine learning system that recommends personalized offers and discounts to customers based on their behavior, product trends, and competitor pricing. It also explains why each recommendation is made using explainable AI techniques.

## Files Included
- app.py → Main Python script for model and prediction
- model.pkl → Trained machine learning model
- personalized_offer_recommendation.ipynb → Jupyter Notebook with full workflow
- shopping_trends.csv → Dataset used
- requirements.txt → Required libraries

## Technologies Used
- Python
- Pandas
- Scikit-learn
- SHAP
- Matplotlib

## How to Run

1. Install dependencies: pip install -r requirements.txt
2. Run the Python script: python app.py

## Features
- Personalized offer recommendation  
- Machine learning-based predictions  
- Explainable AI (SHAP) integration  
- Uses customer behavior and market data  

##  Output
- Predicts whether a customer will accept an offer  
- Displays key factors influencing the prediction
